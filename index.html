Simon Says Game

Press any to start the game

How to Play Game :

1. Click anywhere on the screen to begin your Simon Says challenge. The game will start showing a sequence of colors.

2. Pay close attention to which button flashes, this is the first color you need to remember.

3. Click on the button with the same color that just flashed to match the game’s move.

4. Each new level adds one more color to the pattern, making it a bit harder.

5. You must click all the colors in the exact order shown from the very beginning.

6. If you press the wrong color, the screen flashes red, and your game ends.

7. After “Game Over,” click anywhere to restart the game and try to beat your highest level!
